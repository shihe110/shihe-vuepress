## Spring-AOP

## 1. Aop

（Aspect Oriented Programming）面向切面编程，是对面向对象编程思想的一种补充。

常见应用场景：
- 日志
- 事务
- 数据库操作
等

在不改变程序源码的情况下，动态的增强方法的功能。这些场景中有很多模板化的代码，aop可解决这些模板化代码，消除臃肿。

aop常见名词概念

|  概念   | 说明  |
|  ----  | ----  |
| 切点  | 要添加代码的地方 |
| 通知  | 向切点动态添加的代码 |
| 切面 | 切点+通知 |
| 连接点 | 切点的定义 |
